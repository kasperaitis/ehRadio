ESP Web Tool

Flash Firmware

With this tool, you can:

  • Flash firmware
  • Connect device to the Wi-Fi network
  • Visit the device's hosted web interface
  • Access logs and send terminal commands

Flashing Mode

With most ehRadio ESP-based devices, you can just connect by USB and flash.

Some may require special steps to enter flashing mode.

Improv Mode

With this, you can set up Wi-fi configuration, visit the device's WebUI, or add to Home Assistant.

For most devices, you can use Improv mode immediately after flashing. But you may need to reset the device (power-off, power-on).

If yours is special, you will need to reset normally and click 'Connect' again.

If you are having issues seeing the option to setup Wi-fi configuration, try reloading this page. It doesn't matter which firmware is selected to send Wi-fi configuration.

Special Notes

These firmwares are generally meant for pre-built radios. They must match the device's hardware configuration. Check the Releases.
